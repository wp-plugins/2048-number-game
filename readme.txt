2048 is a number combination game with the aim to achieve 2048 tile.

== Description ==
Join the numbers and get to the 2048 tile! This is a number combination game with the aim to achieve 2048 tile. Simply use shortcode [2048] on any of your page or post. The game store high score on user browsers.

**How to play:** Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

Based on the open-source works of [Gabriele Cirulli](http://gabrielecirulli.github.io/2048/)

== Installation ==
1. Install from within WordPress plugin installer, or get from WordPress plugin repository
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Insert shortcode [2048] in any page or post. That's it.
